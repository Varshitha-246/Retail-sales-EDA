# Retail Sales EDA

This repository contains an exploratory data analysis (EDA) of a retail sales dataset.

## 📌 What’s included
- `retail_sales_EDA.ipynb` — the main analysis notebook with:
  - data loading + inspection
  - missing value checks
  - basic visualizations
  - monthly revenue + profit trend analysis

- `sales_data.csv` — the dataset used for analysis.

## 🚀 How to run
1. Create a Python environment (recommended):
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the notebook in VS Code / Jupyter and run the cells.

## 📌 Key sections to review
- **Quick overview**: data shape, types, sample rows
- **Missing values**: checks for nulls
- **Monthly revenue & profit trend**: core business insight
- **Visualizations**: profit vs sales, category sales, correlations
